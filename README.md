# Heart-Disease-prediction
A Machine Learning project for predicting heart disease using patient health data. Includes data cleaning, visualization, feature analysis, Logistic Regression, Random Forest, model evaluation, and patient prediction.
Heart Disease Prediction using Machine Learning

This project predicts the likelihood of heart disease based on patient medical attributes such as age, cholesterol, blood pressure, and heart rate.

Features:
• Data Cleaning & Preprocessing
• Exploratory Data Analysis (Heatmap, Bar Graphs)
• Logistic Regression Model (82% Accuracy)
• Random Forest Model (88.5% Accuracy)
• Model Evaluation using Accuracy, Classification Report, and Confusion Matrix
• New Patient Prediction System

Technologies Used:
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
